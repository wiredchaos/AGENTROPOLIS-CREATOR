# GitLawb Zero Local Coding Lane

GitLawb Zero is tracked as an optional sovereign coding lane for the Agentropolis Construction District.

Public positioning:

```text
GitLawb Zero
= local coding agent lane
= your model, your machine, your rules
= operator controlled code execution surface
```

## Role in AGENTROPOLIS-CREATOR

AGENTROPOLIS-CREATOR is the foundry. GitLawb Zero can support the foundry as a local builder execution lane for code, package, and workflow candidate generation.

It does not replace Hermes, OpenWorks Lab, District Exchange, Blender, Unreal Engine MCP, or the verification pipeline.

It sits here:

```text
Operator intent
  -> HERMES / OpenWorks command layer
  -> Model Exchange / local model selection
  -> GitLawb Zero local coding lane
  -> repo patch or package candidate
  -> tests / validation
  -> verifier skills
  -> District Exchange candidate
  -> governed runtime adoption
```

## Why It Matters

The Construction District should remain provider neutral and machine sovereign.

A local coding lane gives the foundry a way to generate and modify code without binding every build task to one cloud IDE, one proprietary agent, or one model provider.

The alignment is:

- local execution first
- model choice by policy
- repo changes as reviewable patches
- tests before adoption
- receipts before promotion
- package candidates before runtime use

## Safe Scope

Allowed public scope:

- document local builder patterns
- define repo patch flows
- define package candidate flows
- define test and verification expectations
- define operator approval gates

Do not expose:

- private routing prompts
- private Agentropolis governance internals
- secrets or credentials
- proprietary world forge recipes
- production deployment authority

## Builder District Canon

```text
Hermes routes.
Model Exchange selects.
GitLawb Zero patches locally.
Verifier skills inspect.
District Exchange ships only approved packages.
```

Core rule:

```text
Local code is not production authority.
Local code becomes a candidate.
Candidates must pass verification.
```
