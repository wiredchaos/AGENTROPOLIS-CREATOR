# Hermes Serious Agent Stack

AGENTROPOLIS-CREATOR is the Foundry for repeatable agent behavior.

## Purpose

Turn repeated workflows into skills, templates, diagrams, memory notes, and reviewable packages.

## Seven lanes

1. Brainstorming: design before implementation.
2. Soul Grader: identity before autonomy.
3. Project Memory: repo memory before rediscovery.
4. Agent Memory Wiki: curated operating memory before context resets.
5. Safe Tool Wrapper: structured tool access before freeform actions.
6. Diagram Maker: visible system maps before vague explanations.
7. Skill Factory: reusable skills before repeated manual workflows.

## Repo boundary

This repo should build and package agent capabilities. Runtime execution belongs elsewhere.
