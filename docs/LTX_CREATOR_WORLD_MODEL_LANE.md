# LTX Creator World Model Lane

## Role

LTX is tracked as an optional external world model lane for AGENTROPOLIS-CREATOR.

The lane turns approved city canon, scene contracts, character references, district briefs, and shot grammar into governed simulation previews, cinematic world plates, previs sequences, and model-pack candidates.

It is not a hard runtime dependency and not a separate Agentropolis repo yet.

## Foundry Position

```text
Creator intent
  -> AGENTROPOLIS-CREATOR
  -> scene / world contract
  -> rights and reference check
  -> HERMES Dispatch
  -> AGENTROPOLIS-AGENT-MCP
  -> policy gate
  -> LTX lane when approved
  -> preview / media diff / simulation artifact
  -> triage
  -> delivery packet
  -> receipt
```

## What CREATOR Owns

- world briefs
- district visual packets
- scene contracts
- character and NPC reference roles
- camera language
- continuity ledgers
- render intent
- take triage notes
- delivery and QC packets
- model-pack scouting notes

## Allowed Use Cases

- AGENTROPOLIS district flythroughs
- HERMES-CITY civic shell explainer scenes
- ASBE previs plates
- NPC scene tests
- persistent room and building concept previews
- public-safe motion loops
- owned or licensed cinematic world assets
- physical AI research prototypes

## Do Not Use For

- private client data
- wallet or settlement simulation without governance
- canonical lore overrides
- unrestricted autonomous tool execution
- rights-unclear reference material
- final claims of physics accuracy without validation

CREATOR may use LTX as a Foundry accelerator.

CREATOR still owns the packaging, continuity, review, and delivery logic.
