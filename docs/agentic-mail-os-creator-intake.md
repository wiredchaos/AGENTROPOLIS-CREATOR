# Agentic Mail OS Creator Intake

Status: WATCHLIST / DISTRICT MODULE
Date added: 2026-07-06
Source: https://github.com/cloudflare/agentic-inbox

## Role

Use Agentropolis Agentic Mail OS as the intake layer for creator opportunities, collaboration requests, asset reviews, and production handoffs.

## Suggested inboxes

```text
creator@agentropic.dev
media@agentropic.dev
intake@agentropic.dev
bwb@agentropic.dev
clear@agentropic.dev
```

## Supported workflow

```text
creator email
  -> Agentic Inbox
  -> Email Agent Interface
  -> Creator district triage
  -> draft response / production checklist / asset request
  -> human approval
```

## Allowed by default

- Summarize creator requests
- Extract deliverables, due dates, and asset links
- Draft reviewable replies
- Route media leads to CLEAR or BWB
- Create production checklist language

## Requires approval

- Sending replies
- Accepting paid work
- Sharing attachments externally
- Deleting or archiving creator leads

## Canon lock

Mail is intake and coordination. The Creator district owns production decisions.
