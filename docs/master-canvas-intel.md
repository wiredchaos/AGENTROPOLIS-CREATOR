# Master Canvas Creator Workflow Note

**Source:** `wassermanproductions/master-canvas`  
**Status:** External upstream tool.  
**Creator role:** Visual planning lane for storyboards, prompts, scenes, assets, and exportable production packets.

## Positioning

Master Canvas is a strong candidate for the Creator District planning layer.

It should not be absorbed into AGENTROPOLIS-CREATOR yet. Treat it as an upstream tool creators can use before importing work into Agentropolis workflows.

## Creator use cases

- Build a visual board for an episode, trailer, ad, reel, or short film.
- Group prompts by scene, shot, camera move, asset, and edit note.
- Turn rough concepts into a structured pre-production packet.
- Export boards for review, handoff, archival, or Hermes execution.
- Keep creative planning local-first before agent orchestration begins.

## AGENTROPOLIS-CREATOR mapping

| Creator workflow | Master Canvas assist |
| --- | --- |
| Scene planning | Canvas boards and scene cards |
| Prompt development | Prompt blocks and storyboard notes |
| Asset coordination | Visual references and media notes |
| Client review | Exportable planning documents |
| Agent handoff | Structured packets for Hermes or MCP adapters |

## Integration rule

Use this language:

> Master Canvas can complement AGENTROPOLIS-CREATOR as an external visual planning tool for creator workflows.

Avoid claiming native support until an adapter exists.

## Future adapter idea

A future Creator adapter can import Master Canvas exports and generate:

- Creator project cards
- scene summaries
- shot lists
- prompt vault entries
- asset manifests
- Hermes task packets
- review-ready production briefs

## Canon lock

Master Canvas is not the Creator District.

It is a planning surface that can feed the Creator District when a clean import contract exists.
