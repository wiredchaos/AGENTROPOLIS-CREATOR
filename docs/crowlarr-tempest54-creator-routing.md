# CROWLARR + TEMPEST54 Creator Routing

## Status

AGENTROPOLIS-CREATOR uses CROWLARR as the creator-tool registry and can surface TEMPEST54 only as an authorized lab, training, and audit content workflow.

This repo should not position TEMPEST54 as a public hackbot or unrestricted offensive system.

## Doctrine lock

> CROWLARR indexes the tools. TEMPEST54 deploys the operatives. Evidence Vault preserves the receipts.

## Creator role

AGENTROPOLIS-CREATOR should use CROWLARR to route:

- image tools
- video tools
- audio tools
- prompt engines
- model profiles
- editing workflows
- publishing pipelines
- benchmark demos
- education modules
- security-awareness content

Core UX principle:

> Add a creative tool once. Route it into every creator workflow.

## Safe TEMPEST54 creator use cases

Allowed creator lanes:

- CTF recap videos
- OWASP training explainers
- internal repo audit reports
- smart contract lab walkthroughs
- compliance-friendly security summaries
- benchmark result explainers
- defensive hardening checklists
- educational simulations with toy targets

Do not generate content that encourages unauthorized access, stealth, credential theft, persistence, exploitation of real third-party systems, or evasion.

## Content routing map

| Source | Creator output |
