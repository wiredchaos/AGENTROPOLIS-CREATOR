# Intelligence Pack

The Intelligence Pack is the first Agentropolis workflow package promoted from the Hermes 20 workflow map.

It turns scattered context into usable intelligence with receipts.

## Included Workflows

| Workflow | Role |
| --- | --- |
| Murder Board | Converts messy research piles into source maps, contradiction maps, and decision-ready briefs. |
| Competitor Autopsy | Tears down competitors across websites, docs, GitHub, pricing, changelogs, and public complaints. |
| Daily Opponent | Red-teams a thesis, product plan, article draft, or strategic assumption before execution. |
| RSS Bouncer | Filters public feeds and release streams for relevance instead of raw novelty. |
| Knowledge Archaeologist | Recovers prior decisions, notes, sessions, and context with receipts. |

## Default Posture

This package is read-only by default.

It may inspect approved sources, summarize evidence, produce maps, and draft recommendations. It may not publish, send messages, modify repositories, spend funds, touch wallets, or change production systems without a separate approval-required package.
