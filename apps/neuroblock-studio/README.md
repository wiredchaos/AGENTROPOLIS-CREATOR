# NEUROBLOCK STUDIO

Spatial shot planning for AI-native filmmakers.

## MVP

NEUROBLOCK STUDIO is an original creator-facing planning workspace inside AGENTROPOLIS-CREATOR.

Current MVP features:

- draggable actor, camera, light, prop, and agent markers
- shot cards and readiness scoring
- inspector for scene and shot metadata
- rule-based validation warnings
- Agent Director suggestions
- local browser save and load
- structured brief export
- seeded demo scene

## Run

Open `index.html` in a modern browser.

## Planned integration

```text
NEUROBLOCK STUDIO
  -> AGENTROPOLIS-CREATOR
  -> AGENTROPOLIS-AGENT-MCP for governed agent and model routing
  -> ASBE for production workflow orchestration
  -> AGENTROPOLIS-GAMING-DISTRICT for shared spatial and 3D systems
  -> AGENTROPOLIS-GTM for launch and distribution workflows
```

## Governance

This MVP performs local, rule-based planning only. External model calls, publishing, asset execution, and MCP tool access should remain gated by explicit human review, capability permissions, and execution receipts.
