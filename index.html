Execution Pipeline

Prompt to governed export.

Agents can propose, generate, render, verify, and package scenes or media without bypassing human control.

  1. 01Prompt, image, video, or session memory
  2. 02Hermes planner routes the request
  3. 03NemoClaw drafts the build plan
  4. 04MCP router selects tool adapter
  5. 05World or media diff preview
  6. 06Verifier plus human approval gate
  7. 07Export package and memory writeback

Anti-Moloch Layer

No direct chaos into production.

No unclear-license assets No unapproved likeness exports No unbounded tool loops No world mutation without diff preview No public render without approval No memory writeback without receipts
