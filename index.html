Execution pipeline

Prompt to governed world mutation.

The repo is designed for agents that can propose, generate, verify, and package 3D scenes without bypassing human control.

  1. 01Prompt, image, video, or session memory
  2. 02Hermes planner and NemoClaw builder
  3. 03MCP router selects tool adapter
  4. 04World or media diff preview
  5. 05Verifier plus human approval gate
  6. 06Export, deploy, and memory writeback
